# MSc Thesis - Computer Engineering

This repository contains the source code for my MSc thesis in Computer Engineering at the University of Padua.

## Compilation

To compile the thesis PDF, run:

```bash
bash run.sh
```

After compilation, the PDF will be available at `output/main.pdf`. Auxiliary build files and logs are stored in `output/logs/`.

**Note:** This requires Docker to be installed on your system.
